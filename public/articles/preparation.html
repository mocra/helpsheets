Preparing for the course

The software we will be using

We have compiled a list of software that we will be using over the course of the next few weeks and hopefully if you will continue to use it for many years to come. If you have any problems installing any of the software below please contact us prior to the course so that we can sort out any problems before we get started and don’t waste any time on the day.

This guide has been broken up into three sections:

OS X, Windows and Unix/Linux

OS X

  1. Ruby 1.8.6/1.8.7
    Ruby on OS X comes with the developer tools that can be found on your installation disk.
  2. RubyGems
    RubyGems is the standard Ruby package manager. You should already have this as it comes with OS X but you will want to update it:
    $ gem update --system
$ gem -v
1.3.5
  3. Rails 2.3.5
    You are now ready to install rails, open up terminal and type the following:
    $ gem install rails
$ rails -v
2.3.5
  4. Git
    The easiest way to get Git on OS X is using the git-osx-installer, download your copy from http://git-osx-installer.googlecode.com/files/git-1.7.0.3-intel-leopard.dmg and install.
  5. A syntax highlighting text editor
    This is what you are going to be spending most of your time in so use something you are comfortable with that also has good syntax highlighting support.
    1. Textmate (Everyone at Mocra uses (and loves to use) Textmate and I highly reccomend giving it a go.)
    2. RadRails
    3. Vim
  6. Screensharing
    To turn on screensharing in OS X go into your ‘System Prefrences’, click ‘Sharing’ and tick ‘Screen Sharing’, make sure you also tick ‘Anyone may request permission to control screen’ under ‘Computer Settings’.

Windows

“Instant Rails is a one-stop Rails runtime solution containing Ruby, Rails, Apache, and MySQL, all pre-configured and ready to run. No installer, you simply drop it into the directory of your choice and run it. It does not modify your system environment.”

  1. Ruby/Rails/Apache/MySQL
    Instant Rails http://rubyforge.org/projects/instantrails/.
  2. Git
    The instructions for PC can be found here: https://git.wiki.kernel.org/index.php/Installation.
  3. A syntax highlighting text editor
    This is what you are going to be spending most of your time in so use something you are comfortable with that also has good syntax highlighting support.
    1. E Text Editor (Supports textmate bundles)
    2. InType
    3. Notepad++
    4. RadRails
    5. Vim
  4. Screensharing
    To set up screensharing head over to the TightVNC website and download the 2.0 Beta 1 and install, when it asks you for a Primary password set it to ‘mocra’. You will need to keep this app running during the Mocra Ruby on Rails training course.
