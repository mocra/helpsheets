Ruby - Basics

To show off the basics of ruby you can jump straight into the ruby console. Do so by opening terminal (or the console2 program we installed earlier if you’re on a PC) and typing:

irb

Basic Ruby Syntax

Sub heading

def hello_world
  puts "Hello!"
end
#=> nil
hello_world
#=> Hello!

Sub heading

def hello_world(name)
  puts "Hello #{name}!"
end
#=> nil
hello_world("World")
#=> Hello World!
#=> nil

Sub heading

def hello_world(name = "World")
  puts "Hello #{name.capitalize}!"
end
#=> nil
hello_world "odin"
#=> Hello Odin!
#=> nil
hello_world
#=> Hello World!
#=> nil

Sub heading

class Greeter
  def initialize(name = "World")
    @name = name
  end
  def say_hi
    puts "Hi #{@name}!"
  end
  def say_bye
    puts "Bye #{@name}, come back soon."
  end
end
#=> nil
